# Counter
Coding with the students on Week 7 at Boca Code. 
Class titled "BOM (Browser Object Model) and DOM (Document Object Model)".

## Parameters
- Increase counter.
- Decrease counter.
- Reset the counter to zero.
- If counter goes below zero, redner it as zero.
